## Функционал прльзователя
Пользователю отображаются все акции, услуги и фото данного салона красоты. Функционал:
* создать аккаунт 
* онлайн запись
  > онлайн запись доступна только после регистрации
* отмена записи 
* звонок менеджеру 
* перейти в WhatsApp
* перейти в Instagram
* перейти на сайт 
* добавить в избранное
